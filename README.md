# 📊 ПП-дашборд

Онлайн-мониторинг состояния платежей. Обновляется каждые 30 минут по будням.

## Секции дашборда

1. **Сделки текущего дня** — поступления рублей, отправки на VDX/Almena, свежие заявки
2. **Сделки прошлых дней** — сделки, требующие внимания (ждут зелёнку, ПП или заявку)

## Данные

Дашборд читает 5 таблиц:
- `data.json` — Main
- `data_konstantin.json` — Константин (Просто группа)
- `data_molodezh.json` — Молодёжь
- `data_dvizhenie78.json` — Движение 78
- `data_flowers.json` — Цветы

## Разработка

```bash
node build-dashboard.js
# → генерирует index.html
```
